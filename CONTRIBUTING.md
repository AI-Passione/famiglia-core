# Contributing to the Famiglia

Welcome to the **Famiglia Core**. We are building the most elite, autonomous, and beautiful multi-agent ecosystem in the world: **"AI Passione."**

In this house, we value loyalty, precision, and a **"Vibe First"** approach. Follow the path below to become part of the Core.

---

## 🤝 The Rule of the House: Talk First

The Core does not like surprises. Before you code:
1.  **Start a Discussion**: For new ideas or visions, use the [Discussions](https://github.com/AI-Passione/famiglia-core/discussions) tab.
2.  **Open an Issue**: For bug reports or specific tasks, use the [Issues](https://github.com/AI-Passione/famiglia-core/issues) tab.

**Do not open a Pull Request without an associated Issue or a blessed Discussion.**

---

## 🚀 The Path of a Contribution

1.  **Fork**: [Fork](https://github.com/AI-Passione/famiglia-core/fork) the repository. We do not work on core branches.
2.  **Branch**: Use `feat/`, `fix/`, or `docs/` prefixes.
3.  **PR**: Open a PR into `main`. Use the [Template](.github/PULL_REQUEST_TEMPLATE.md) and link the Issue.
4.  **Review**: A Consigliere will review. If it doesn't "WOW" us, be ready for adjustments.

---

## 🎨 Development Standards

### 🌊 Vibe Compliance (The "Silent Concierge")
All UI must adhere to **Modern Italian Noir**:
*   **Tonal Layering**: Use background shifts, not solid borders.
*   **Glassmorphism**: 60% opacity + 20px backdrop-blur for agent modules.
*   **Typography**: *Noto Serif* (Headers), *Manrope* (Body), *Space Grotesk* (Metadata).
*   **Aesthetics**: Charcoal and Burgundy palette with Gold accents.

### Python & Frontend
*   **Python**: Use `uv` for management. Follow PEP 8 and `ruff`.
*   **Frontend**: React with Vite. Stick to the Vibe Coding Manifesto.
*   **Releases**: Merges to `main` trigger `:staging` builds. Production is manually triggered via Actions.
