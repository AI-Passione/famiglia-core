# Slack Channel Structure

## **Slack Channel Structure**

### **🔴 HIGH PRIORITY (Don Jimmy Monitors)**

**`#command-center`**

- Don Jimmy issues orders
- Agents acknowledge tasks: "Capito, Don Jimmy" "Fatto, Boss"
- Alfredo coordinates task delegation (with approval in Phase 1)
- **Notifications:** ✅ ON for Don Jimmy

**`#alerts`**

- 🚨 URGENT items only (budget overruns, system failures, critical blockers)
- Vito: Suspicious transactions, budget alerts
- Riccado: Pipeline failures, infrastructure outages
- Tommy: Critical execution blockers
- Alfredo/Bella: Daily/weekly alert summaries
- **Notifications:** ✅ ON for Don Jimmy

**`#daily-brief`**

- Dr. Rossini: Morning research/strategy updates (Mon-Fri 9am)
- Vito: Market conditions, portfolio status (Mon-Fri 8am)
- Alfredo: Agent status, coordination summary (daily)
- **Notifications:** ❌ OFF (Don Jimmy checks during morning routine)

**Notifications:** ✅ ON for Don Jimmy
